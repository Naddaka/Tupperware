Свойства товаров

Шаг 1

Чтобы создать новое свойство или редактировать уже существующее, необходимо перейти: Каталог товаров → Свойства товаров

Шаг 2

Для создания нового свойства следует кликнуть по кнопке "Создать свойство".

Для редактирования свойства необходимо кликнуть по названию свойства в столбце "Свойство".

Шаг 3

Указать или изменить содержимое полей:

  • Имя
  • Имя колонки CSV (данное имя будет использовано при импорте нужного значения свойства в товар)
  • Использовать в категориях
  • Значение (если свойство будет иметь множественный выбор, то данный пункт обязателен к заполнению)
  • А также осуществить настройку создаваемого свойства в блоке ячеек.

Сохраните изменения при помощи кнопки "Сохранить" или "Сохранить и выйти" в правом верхнем углу экрана

Шаг 4

Присвоить товарам (Административная панель→каталог товаров → список товаров → нужный товар → свойства) для соответствующего свойства нужные значения свойства.

Шаг 5
