Быстрое наполнение

Быстрое создание категории товара

В панели администратора перейдите: Каталог товаров → Категории товаров → клик по кнопке "Открыть быстрое создание".

Заполните необходимые поля и нажмите кнопку "Создать".

Быстрое создание товара.

В панели администратора перейдите: Каталог товаров → Список товаров → клик по кнопке "Открыть быстрое создание".

Заполните необходимые поля и нажмите кнопку "Создать".

Быстрое создание свойства товара.

В панели администратора перейдите: Каталог товаров → Свойства товаров → клик по кнопке "Открыть быстрое создание".

Заполните необходимые поля и нажмите кнопку "Создать".

Быстрое создание бренда.

В панели администратора перейдите: Каталог товаров → Бренды → клик по кнопке "Открыть быстрое создание".

Заполните необходимые поля и нажмите кнопку "Создать".
