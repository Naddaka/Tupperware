Работа с баннерами

Что изменится на сайте

Главная страница.

Система содержит модуль xbanner, который по умолчанию совместим только с шаблоном FullMarket, все остальные шаблоны работают с модулем Banners. О нем пойдет речь дальше.

Шаг 1

Модули → Управление баннерами

Шаг 2

В открывшемся окне вы можете:

Создать новый баннер, осуществив клик по кнопке "Создать баннер".

Редактировать существующий баннер, кликнув по названию баннера.

Шаг 3

Указать или изменить демо-данные в формах:

  • Название
  • Активный
  • URL (адрес страницы, на которую будет вести баннер)
  • Баннер группы — в данном окне надо создать и/или выбрать (отметить) группы для вывода боковых баннеров (названия группы можно найти в файле \templates\ВашШаблон\shop\start_page.tpl после getByGroup )
  • Выбранные элементы — здесь надо отметить выбранную категорию из следующего пункта (main-all, например)
  • Показать в категориях (выбранные элементы) — выбрать нужную сущность баннера. Она должна вывестись в "Выбранные элементы"
  • Изображение
  • Активен до (указать обязательно).

Шаг 4

Сохранить изменения при помощи кнопки "Сохранить" или "Сохранить и выйти"

Важно: на странице со списком баннеров должна быть активирована опция "Настройки шаблона" - Используйте различные шаблоны для разных страниц.

Шаг 5

Очистите кеш интернет-магазина, осуществив переход: Настройки → Очистить кеш:
