Работа с комментариями

Шаг 1

Модули → Комментарии

Шаг 2

Для модерации комментариев необходимо:

Кликнуть по ID комментария

При необходимости редактируем:

  • Имя пользователя
  • Е-mail пользователя
  • Статус
  • Текст комментария

Для настройки необходимо:

Кликнуть по кнопке "Настройки", , здесь также можно задать использование защитного кода (каптча), включить одобрение комментариев администратором, ограничить комментарии по количеству символов и частоте комментирования одним пользователем, запретить комментирование для незарегистрированных пользователей.

Указываем данные в форме:

  • Максимальная длина комментария
  • Частота комментирования
  • Отмечаем необходимые пункты в блоке ячеек

Шаг 3

Сохранить изменения при помощи кнопки "Сохранить" или "Сохранить и выйти" в правом верхнем углу экрана

Чтобы изменения вступили в силу незамедлительно, необходимо очистить кэш из административной панели сайта: Настройки → очистить кэш
