Импорт-экспорт товаров

Импорт и экспорт товаров позволяют администратору магазина переносить большие объемы информации без существенных затрат времени и труда. В ImageCMS Shop импорт и экспорт данных осуществляется при помощи одного функционального окна.

Форматы файлов, которые поддерживает система ImageCMS Shop, — CSV, XLS или XLSХ.

Чтобы открыть окно обмена данными

выберите пункт в административной панели пользователя "Модули" (список всех модулей), а в нем — подпункт "Импорт-экспорт CSV/XLS":

Элементы функционального окна "Импорт-экспорт CSV/XLS"

Данное окно содержит три вкладки: "Импорт", "Экспорт" и "Список архивов экспорта". Следовательно, функциональные элементы, доступные пользователю на той или иной вкладке, отличаются друг от друга:

Функциональные элементы вкладки "Экспорт":

  1. Блок ячеек "Столбцы" предоставляет возможность выбрать данные для товаров, подлежащих экспортированию. Каждая ячейка, имеющая значение "Выбрано", будет отображена в создаваемом файле отдельной колонкой, которая содержит соответствующие данные.
  2. Категории — этот элемент предоставляет возможность выбрать категории, подлежащие экспортированию. Из данного списка можно выбрать все необходимые категории при помощи зажатой клавиши Ctrl. Если не выбрать ничего, тогда экспортируются все товары, доступные на сайте.
  3. Кнопка "Просмотреть выбранные свойства" — функциональный элемент, запускающий процесс вывода блока свойств (параметров) товаров для экспорта значений данных свойств.
  4. Свойства товаров (параметров) — это блок ячеек, определяющий параметры товаров, которые содержатся в выбранной категории и подлежат экспортированию. Ячейки отображают те параметры, которые были созданы и присвоены администратором при помощи пункта меню "Свойства товаров".
  5. Группа чекбоксов "Тип файла" предоставляет возможность выбрать формат создаваемого файла экспорта.
  6. Кнопка "Сделать резервную копию фотографий" предоставляет вышеуказанную возможность.
  7. Кнопка "Начать экспорт" — функциональный элемент, запускающий процесс экспорта данных из ImageCMS Shop в файл.

Функциональные элементы вкладки "Импорт":<

  1. Кнопка "Выберите файл" предоставляет администратору возможность выбрать файл, содержащий данные для импорта.
  2. "Язык" — выбор локализации, в которую осуществляется импорт (полноценная мультиязычность доступна в Premium-пакетах магазина).
  3. Блок слотов для файлов — предоставляет возможность администратору загрузить до 3-х файлов и импортировать их в нужное время (файлы в слотах можно заменить иными файлами импорта).
  4. "Свойства" — ячейки, в которых отображаются категории данных, содержащиеся в загруженном в слот файле импорта, которые будут импортированы на сайт.
  5. Блок ячеек выбора — предоставляет администратору возможность опционально осуществлять ряд вспомогательных импорту действий, таких как: обновлять пустые поля (перезапись существующих данных пустыми из файла импорта), сделать снимок базы данных до начала импорта (резервная копия базы данных), начать изменения размера изображения после импорта (при импорте новых изображений) и обновлять цены после импорта (при изменении цен на сайте импортом).
  6. Кнопка "Начать импорт" — функциональный элемент, непосредственно запускающий процесс импорта данных в систему:

Примечание: надпись "Пропустить колонку" свидетельствует о том, что импорт этих данных проведен не будет (нарушена пунктуация (структура) в файле импорта, или система не содержит данных для внесения записи).

Примечание: столбцы name, num, cat, являются обязательными. Их наличие и содержание необходимо для выполнения импорта системой.

После успешного осуществления вышеуказанных действий, процесс импорта происходит автоматически и не требует вмешательства администратора.

Импорт изображений

В системе ImageCMS Shop предусмотрена возможность импорта изображений через URL-адрес изображения и непосредственно из папки системы на сервере.

Импорт изображений при помощи URL

Данный способ будет весьма полезным и удобным при импорте небольшого количества товаров.

Чтобы импортировать изображения при помощи URL, вам необходимо открыть созданный файл импорта.

Файл импорта должен содержать колонки:

  • vimg - колонка для главного изображения;
  • imgs - колонка для дополнительных изображений.

В этих колонках необходимо указать:

  • URL главного изображения товара в колонке vimg;
  • URL дополнительного изображения товара в колонке imgs. Если дополнительных изображений несколько, вам необходимо в одной ячейке колонки указать их URL (отдельно для каждого изображения), используя как знак разделения вертикальный слеш (без пробелов):

http://example.com/image/path/01.jpg | http://example.com/image/path/02.jpg | http://example.com/image/path/03.jpg | ...

Узнать URL нужного вам изображения можно при помощи функции вашего браузера "Просмотр элемента", обычно доступной при нажатии клавиши F12.

Импорт изображений из папки системы на сервере.

Основные фото товаров надо помещать в папку системы на сервере /uploads/origin, а дополнительные фото — в /uploads/origin/additional.

На папку uploads рекурсивно (включая вложенные папки и файлы) должны стоять права на запись/чтение 777.

Примечание: названия изображений должны содержать только латинские буквы и быть написаны без пробелов.

Осуществляя импорт, обязательно выберите ячейку "Начать изменения размера изображения после импорта", находящуюся на вкладке "Импорт". В противном случае импортированные изображения товаров отображаться не будут.

Функциональные элементы вкладки "Список архивов экспорта":

На этой вкладке выводятся информационные данные с перечнем резервных копий баз данных, которые были созданы при импорте, если была выбрана опция "сделать снимок базы данных до начала импорта".

Данные архивы можно скачать на локальный ПК или же удалить.p>

Важно: на папку \application\backups (включая вложенные папки и файлы) должны рекурсивно стоять права на запись/чтение 777, в противном случае, импорт и вывод списка архивов импорта будут невозможны.
