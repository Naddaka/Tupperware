Скидки

Чтобы создать/редактировать скидку, необходимо:

Шаг 1

В панели администратора перейти: Модули → Скидки интернет-магазина

Шаг 2

Для создания скидки нужно кликнуть по кнопке "Создать скидку".

Для редактирования скидки кликните по названию скидки в столбце "Ключ".

Сам код скидки может быть изменен/создан только с помощью случайной генерации.

Шаг 3

В открывшейся форме создания/редактирования скидки необходимо:

  • Для создания скидки:
    1. Указать название скидки
    2. При необходимости установить ограничение на количество применений (каждый подарочный сертификат ограничен 1 использованием)
    3. Выбрать метод подсчета скидки
    4. Выбрать тип скидки
    5. При необходимости указать временной интервал действия скидки.

  • Редактированию подлежат поля:
    1. Название скидки
    2. Код скидки в системе (автогенерация)
    3. Ограничение на количество использования
    4. Метод подсчета скидки, суммы или процента
    5. Тип скидки
    6. Временной интервал действия скидки.

Примечание: вы можете создать скидку для каждой языковой версии магазина при помощи переключателя языковой версии, который доступен в форме редактирования скидки. Но имейте ввиду, что данный функционал есть только в версии Premium.

Шаг 4

Сохраните изменения при помощи кнопки "Создать" или "Сохранить и выйти" в правом верхнем углу экрана.

Чтобы изменения вступили в силу незамедлительно, необходимо очистить кэш из административной панели сайта: Настройки → Очистить кэш
