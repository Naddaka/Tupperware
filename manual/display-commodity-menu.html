Меню категорий товаров

Для настройки отображения меню следует:

Шаг 1

В панели администратора перейти: Модули → Управление дизайном → Вкладка «Колонки меню».

Шаг 2

В открывшейся вкладке следует настроить отображение меню, оперируя значениями:

  1. Поля отображения уровней
  2. Колонки "Отображать в колонке"

Для отображения меню в виде колонок следует задать значения:

  • Поля отображения уровней — значение «Все категории»
  • Колонки «Отображать в колонке» — значение «0».

Для отображения меню в виде дерева категорий задайте значения:

  • Поля отображения уровней — значение «Категории второго уровня»
  • Колонки «Отображать в колонке» — значение «0».

Для отображения меню в виде дерева с колонками категорий задайте значения:

  • Поля отображения уровней — значение «Категории второго уровня»
  • Колонки «Отображать в колонке» — числовые значения от 0 до 10.

Примечание: опция отображения «Категории второго уровня» не сохранится, если нет ни одной категории 3-го уровня вложенности (надо добавить в Административной панели→Каталог товаров→Категории товаров→Создать категорию). Это можно сделать хотя бы временно — для активации настройки, затем можно удалить/деактивировать.

Шаг 3

Сохраните изменения при помощи кнопки «Сохранить» в левом нижнем углу экрана.

Чтобы изменения вступили в силу незамедлительно, необходимо очистить кэш из административной панели сайта: Настройки → Очистить кэш
