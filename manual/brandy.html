Бренды

Что изменится на сайте

Главная страница.

Страница брендов.

Шаг 1

Для создания бренда необходимо кликнуть:

Каталог товаров → Бренды → Клик по кнопке «Создать бренд».

Для редактирования бренда необходимо кликнуть:

Каталог товаров → Бренды → Клик по названию бренда, подлежащего редактированию.

Шаг 2

В открывшихся формах редактирования/создания следует указать корректные данные в полях:

  • Название
  • URL (заполняется автоматически, согласно названию бренда)
  • Описание
  • Мета-данные (можно не заполнять на первом этапе настройки бренда)
  • Изображение

Шаг 3

Сохраните изменения при помощи кнопки «Сохранить» или «Создать» в правом верхнем углу экрана.

Чтобы данный функционал работал корректно, необходимо:

  • Удалить все демо-товары (если демо-данные присутствуют на сайте)
  • При создании актуальных товаров присваивать им соответствующий бренд в поле “Название бренда”

Шаг 4

Для редактирования виджета брендов на главной странице сайта перейдите: Настройки → Виджеты → Brands → Гаечный ключ

Здесь можно редактировать количество брендов для отображения и вариант вывода брендов (только с фото, или же всех существующих).

Вносите необходимые настройки, после сохранения изменений необходимо очистить кэш: Настройки → Очистить кэш
