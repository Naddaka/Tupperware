Количество товаров

Чтобы настроить количество выводимых товаров, необходимо:

Шаг 1

В панели администратора перейти: Настройки → Настройки магазина → вкладка "Внешний вид"

Шаг 2

В части вкладки "Внешний вид" в соответствующих полях указать:

  1. Количество товаров, отображаемых на одной странице товарной категории
  2. Возможные значения для количества вывода товаров на странице (позже покупатель сможет выводить товар на страницу категории, выбрав значения из поля "На странице")
  3. Перетягивание определенного метода сортировки вверх списка делает его методом сортировки по умолчанию, если не сделаны соответствующие настройки для определенной категории (Административная панель → Каталог товаров → Категории товаров → Редактирование категории). Смотрите подробнее в разделе документации "Сортировка товара".

Шаг 3

Сохраните изменения при помощи кнопки "Сохранить" или "Сохранить и выйти" в правом верхнем углу экрана.

Чтобы изменения вступили в силу незамедлительно, необходимо очистить кэш из административной панели сайта: Настройки → очистить кэш
