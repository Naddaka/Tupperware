Пользователи

Чтобы создать пользователя или группу пользователей (роль), необходимо:

Шаг 1

В панели администратора перейти: Пользователи → ...

  • Для пользователя - ... → Список пользователей

  • Для группы (роли) - ... → Управление правами доступа

Шаг 2

В открывшейся форме необходимо:

Для пользователя

1. Кликнуть по кнопке «Создать пользователя»

2. В открывшейся форме внести изменения в соответствующие поля:

  • E-mail (обязательно)
  • Пароль (обязательно)
  • Роль
  • Полное имя
  • Телефон (обязательно)
  • Адрес

3. Осуществить клик по кнопке "Новая роль"

4. В открывшейся форме внести изменения в соответствующие поля:

  • Имя
  • Описание
  • Важность

5. Осуществить выбор доступных прав для группы на вкладках:

  • Магазин
  • Базовые
  • Модули

Примечание: вкладка «Магазин» доступна только в пакете Premium. Поэтому ограничение пользователей в административных правах (создание менеджеров для наполнения магазина с ограниченными правами, например) доступно только для интернет-магазинов на версии Premium.

Шаг 3

Сохраните изменения при помощи кнопки «Создать» или «Создать и выйти» в правом верхнем углу экрана.
