Работа с товарами

Шаг 1

Каталог товаров → Список товаров

Для создания нового товара кликните "Создать товар".

Для редактирования - в соответствующем столбце кликните по названию товара, подлежащего редактированию.

Шаг 2

В форме создания нового товара вносим основные данные в раздел "Информация" по полям:

  • Название товара (при необходимости - название варианта)
  • Цена
  • Артикул товара (обязательно, если планируете использовать функционал Экспорта/Импорта товаров)
  • Количество единиц товара в наличии
  • Валюта
  • Загрузить изображение
  • Бренд
  • Главная товарная категория (при необходимости - дополнительные товарные категории)
  • Краткое и полное описание

Шаг 3

Сохраните изменения при помощи кнопки "Сохранить" или "Создать" в правом верхнем углу экрана.

Шаг 4

Указать/изменить свойства товара на вкладке формы редактирования "Свойства".

Перед тем, как присваивать товару значения для характеристики, их нужно создать

Загрузить/изменить дополнительные изображения.

Шаг 5

Сохраните изменения при помощи кнопки "Сохранить" или "Создать" в правом верхнем углу экрана.

Если вам нужно заполнить только основные данные без расширенных настроек, рекомендуем попробовать функцию быстрого наполнения - быстрое создание товара
