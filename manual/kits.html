Комплекты

Для создания нового набора товаров или редактирования уже существующего, необходимо:

Шаг 1

В панели администратора перейти: Каталог товаров → Наборы товаров → ...

Для создания нового комплекта:

... → Кликнуть по кнопке "Создать набор"

Для редактирования существующего набора:

... → Кликнуть по уникальному идентификатору набора в столбце "ID"

Шаг 2

В открывшейся форме создания/редактирования в соответствующих полях следует:

Для создания набора:

  1. Указать главный товар в поле "Главный товар"
  2. Указать дополнительный товар в поле "Связанные продукты"
  3. Указать процент скидки на дополнительный товар
  4. Сделать выбор в ячейке "Активный"

Для редактирования набора:

  1. При необходимости внести изменения в поле "Главный товар"
  2. При необходимости внести изменения в поле "Связанные продукты"
  3. При необходимости внести изменения в поле "Скидка %"
  4. При необходимости внести изменения в ячейке "Активный"

Шаг 3

Сохранить изменения при помощи кнопки "Сохранить" или "Создать" в правом верхнем углу экрана

Чтобы изменения вступили в силу незамедлительно, необходимо очистить кэш из административной панели сайта: Настройки → Очистить кэш
