Варианты товара

Функционал, предусмотренный для товаров, которые имеют несколько характеристик (например: цвет, размер, объем).

Чтобы добавить вариант товара, вам необходимо:

Шаг 1

В панели администратора перейти: Каталог товаров → Список товаров → Клик по названию товара, вариант которого вы желаете добавить.

Шаг 2

В открывшейся форме редактирования следует:

  1. Кликнуть по кнопке "Создать вариант"
  2. Указать название варианта товара
  3. Указать цену варианта товара
  4. Указать валюту для цены
  5. Загрузить изображение варианта товара.

Примечание: вкладка "Изображения" становится также доступной во время создания товара (после его сохранения).

Шаг 3

Сохраните изменения при помощи кнопки "Сохранить" или "Сохранить и выйти" в правом верхнем углу экрана.

Чтобы изменения вступили в силу незамедлительно, необходимо очистить кэш из административной панели сайта: Настройки → Очистить кэш
