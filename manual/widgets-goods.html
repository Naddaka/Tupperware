Виджеты товаров

В системе ImageCMS по умолчанию используются 3 виджета товаров, которые отображаются в шаблоне соответственно используемому дизайну:

  • Виджет "Акционные товары"
  • Виджет "Популярные товары"
  • Виджет "Новинки"

Данные виджеты могут быть выведены в разном сочетании, форма и положение виджетов зависит от выбранного шаблона (варианта дизайна).

Шаг 1

В панели администратора перейти: Каталог товаров → Список товаров

Шаг 2

Чтобы вывести ваши актуальные товары через виджеты, в открывшемся списке следует присвоить товарам соответствующие отметки в колонке "Статус"

Отметка для акционных товаров

Отметка для популярных товаров

Отметка для новинок

Примечание: присвоить отметку товару можно непосредственно в форме создания товара (после его сохранения).

Шаг 3

Очистите кэш интернет-магазина: Настройки → Очистить кеш

Шаг 4

Затем нужно зайти в Административную панель→ Настройки→ Виджеты→ Необходимый виджет→ Пиктограмма гаечного ключа→ сделайте требующиеся настройки (количество выводимых товаров в виджете, товары с определенным статусом, заголовок виджета). В некоторых случаях заголовок можно сменить через модуль "Переводчик".
