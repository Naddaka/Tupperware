Работа с категориями

Главная страница

Шаг 1

Чтобы создать новые или отредактировать уже созданные товарные категории, следует:

  • Для создания категории перейти: Каталог товаров → Категории товаров → Клик по кнопке «Создать категорию».

  • Для редактирования категории перейти: Каталог товаров → Категории товаров → Клик по названию категории, подлежащей редактированию.

Шаг 2

В открывшейся форме редактирования/создания следует указать/изменить соответствующие данные в полях:

  • Название
  • Родитель (для подкатегории)
  • URL (создается автоподбором, можно не изменять)
  • Сортировка по умолчанию (вариант сортировки товаров в категории).

Важно: вариант отображения категории зависит от выбранного шаблона категории и уровня вложенности категории.

Для вывода товаров в категории нужно задать шаблон вывода category (т.е. для активации шаблона по умолчанию надо оставить данное поле пустым). Для вывода вложенных подкатегорий служит шаблон вывода categorysubsecond, а для вывода соседних подкатегорий — шаблон вывода categorysubfirst.

Шаг 3

Сохраните изменения при помощи кнопки «Сохранить» или «Сохранить и выйти»

Чтобы изменения вступили в силу незамедлительно, необходимо очистить кэш из административной панели сайта: Настройки → Очистить кэш
