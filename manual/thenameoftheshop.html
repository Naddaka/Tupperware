Название магазина

Что изменится на сайте

Заголовок сайта.

"Подвал" сайта.

Шаг 1

Для изменения "подвала" сайта:

Настройки → Глобальные настройки → Информация о сайте → Название сайта (тип предприятия)

Форма и положение данного элемента зависит от выбранного шаблона (варианта дизайна).

Шаг 2

Для изменения заголовка необходимо перейти:

Настройки → Глобальные настройки → Заполнить Мета-теги → Название сайта

Шаг 3

Сохраните изменения при помощи кнопки "Сохранить".

Для того, чтобы изменения вступили в силу незамедлительно, необходимо очистить кэш из административной панели сайта: Настройки → Очистить кэш
